# フォルクローレ楽曲分析リポジトリ
昔から、リズムの似ているSanjuanitoとTobasのメロディ的な違いが気になってて、いろいろ考えて分析してます。
とりあえず少量のデータでやってみたのはこちら。
https://www.notion.so/313fb3bb4890808d8a36c3a7e6810c84?source=copy_link

- 奏者たちは、なんとなくメロディ的な違いを感じ取ることができます。聞いてると「これはエクアドルっぽい」「これはボリビアっぽい」と、なんとなくわかります
- 先輩から、「実はSanjuanitoとTobasは、パーカッションの使い方が違う」といった意見をいただいています

# これからやるべきこと
- 各楽曲の原作者の紐づけ
- タイトルベースで重複を消す。原作者でないものは落とすこと（創作年度なども大事）
- サンファニートはエクアドルのもののみ、トバスはボリビアのみとする
- まずは30曲~50曲ずつ絞る
- その中から、またさらに各リズムからそれているものを落とす
- 最終的に30曲ずつ確保しておく。Amazon Musicにファイルがなかったり、分析がうまくいかないこともあるかもしれないので多めにとっておこう
